# Publication scope

This repository is a sanitized public mirror.

Excluded intentionally:

- original git history;
- `.env` files and runtime secrets;
- SQLite databases, backups, WAL/SHM files;
- private deployment scripts and GitHub Actions workflows;
- host-specific deployment docs;
- logs and local agent/session state;
- local-only diagnostic shell scripts.

Included intentionally:

- source code for API, web, WhatsApp assistant, shared types, migrations, tests;
- sanitized env examples;
- local development commands;
- portfolio README and deployment overview;
- AI-agent eval/bench source where safe.
